# Twirmenal

A  simple Twitter Terminal app

## Installation

Add this line to your application's Gemfile:

    gem 'twirmenal'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install twirmenal

## Usage

After installing it you can run it with

    $ twirmenal
